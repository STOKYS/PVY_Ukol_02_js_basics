5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, += (To samé jako x = x + y), -= (To samé jako x = x - y), *= (To samé jako x = x * y), /= (To samé jako x = x / y), %= (To samé jako x = x % y), **= (To samé jako x = x ** y)

Logické

|| (Logicke OR), && (Logicke AND), ! (Logicke NOT)

Porovnávací

<, >, <= (Menší nebo rovná se), >= (Větší nebo rovná se), ==

Bitové

& (AND), | (OR), ~ (NOT), ^ (XOR), << (Posunutí nuly do leva), >> (Podepsaný pravý posun), >>> (Posunutí nuly do prava);

Typové

typeof (typ proměnné), instanceof (Vrátí true, pokud objekt je instancí typu objektu)
